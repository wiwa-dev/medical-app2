# CMSFP - Projet de Gestion Financière Medicale ## Architecture Technique: Wails (Go) + React ### 1. Modèle de Données (SQLite Schema) ```sql -- Table des recettes par service CREATE TABLE daily_entries ( id INTEGER PRIMARY KEY AUTOINCREMENT, date DATE NOT NULL, service_type TEXT CHECK(service_type IN ( 'Analyses biologiques', 'GSRH', 'ECG', 'Echocœur', 'Holter ECG', 'MAPA', 'Dentiste', 'Ophtalmologie', 'Imagerie médicale', 'Cardiologie' )), amount INTEGER DEFAULT 0, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); -- Table des dépenses journalières CREATE TABLE daily_expenses ( id INTEGER PRIMARY KEY AUTOINCREMENT, date DATE NOT NULL, description TEXT NOT NULL, amount INTEGER DEFAULT 0, created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP ); -- Table des honoraires mensuels CREATE TABLE monthly_honoraires ( id INTEGER PRIMARY KEY AUTOINCREMENT, month INTEGER NOT NULL, -- 1-12 year INTEGER NOT NULL, person_name TEXT NOT NULL, role TEXT, amount INTEGER DEFAULT 0 ); -- Table des investissements CREATE TABLE investments ( id INTEGER PRIMARY KEY AUTOINCREMENT, month INTEGER NOT NULL, year INTEGER NOT NULL, description TEXT, amount INTEGER DEFAULT 0 ); -- Table des analyses gratuites CREATE TABLE free_analyses ( id INTEGER PRIMARY KEY AUTOINCREMENT, month INTEGER NOT NULL, year INTEGER NOT NULL, category TEXT CHECK(category IN ('agent_etat', 'non_agent')), amount INTEGER DEFAULT 0 ); ``` ### 2. Handlers Go (Wails) - Exemples de signatures ```go package main import ( "context" "database/sql" ) type App struct { ctx context.Context db *sql.DB } // Enregistrer les recettes d'une journée func (a *App) SaveDailyReceipts(date string, entries map[string]int) error { // Logique d'insertion SQLite return nil } // Récupérer le bilan mensuel func (a *App) GetMonthlyReport(month int, year int) (ReportData, error) { // Agrégation SQL (SUM) pour les recettes et dépenses return ReportData{}, nil } // Calculer les honoraires (40% ou 60% des recettes du service) func (a *App) CalculateHonoraires(service string, totalReceipts int, percentage float64) int { return int(float64(totalReceipts) * percentage) } ``` ### 3. Logique de Calcul des Règles Métier - **Profit Mensuel** = Somme(Recettes) - (Somme(Honoraires) + Somme(Dépenses)) - **Solde Final** = Solde Précédent + Profit Mensuel - Investissements - **Remise à zéro hebdo** : Les totaux hebdomadaires sont calculés dynamiquement basés sur le `date` du lundi.
